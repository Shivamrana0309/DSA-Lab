# 📘 DSA Classroom Practice – Java

This repository contains my implementations and notes for **Data Structures and Algorithms (DSA)** taught in class. The code is written in **C++** and follows the topics covered during our curriculum.

---

## 🧠 Topics Covered

### 🌳 Trees & Binary Search Trees
- Binary Tree Implementation
- Inorder, Preorder, Postorder Traversals
- Binary Search Tree (BST)
- Optimal Binary Search Tree (OBST)

### 🔐 Hashing
- Hash Tables using Arrays
- Collision Handling (Chaining, Open Addressing)
- HashMap operations

### 🧮 Graph Algorithms
- Dijkstra’s Shortest Path Algorithm
- Prim’s Minimum Spanning Tree (MST)
- Kruskal’s Algorithm for MST

### 🌀 Heaps
- Min-Heap and Max-Heap Implementation
- Heap Sort Algorithm

### 📦 File Handling
- Reading and writing data to files
- Serialization and Deserialization in Java
- Working with `.txt` and `.dat` files for DSA input/output

### 🧬 Huffman Coding
- Frequency table construction
- Huffman Tree generation
- Encoding and Decoding logic
